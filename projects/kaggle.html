Kaggle Competitions

Live leaderboard rankings across ML competitions — rankings fetched daily from the Kaggle API via GitHub Actions.

All projects Home Kaggle profile ↗

How it works

A GitHub Actions workflow runs every day at 04:00 UTC. It calls the Kaggle API, finds my team on each competition leaderboard, and commits the updated ranks and scores to projects/kaggle-scores.json in this repository. The page then reads that file on load — no server, no secrets exposed in the browser.
